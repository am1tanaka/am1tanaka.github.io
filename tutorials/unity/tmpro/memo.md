# ダイナミックフォントの利用について
ダイナミックフォントを使えば予め大きなフォントアセットを作る必要がなくなります。ただし、実行時の処理時間が落ちることと、ttfかotfフォントを含める必要があるのがネックです。ビットマップフォントの場合、使用する一部のフォントだけを生成して使うので印刷物と同等のライセンスで利用できます。それに対してフォントファイルを組み込む場合は再配布や二次配布に該当してしまうため、フリーフォントの中でも使えないものが出てきます。ライセンスを気にせず利用するなら、ビットマップフォントを作成して利用するのが安心です。
