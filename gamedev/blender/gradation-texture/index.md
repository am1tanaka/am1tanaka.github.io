
# やりたいこと

Godotでローポリ系の3Dゲームの開発を目論んでいます。ローポリの見栄えを良くするポイントとして考えているのが、グラデーションとAOだと考えてます。Unityでは、ライトやポストプロセスで味付けができます。それに対して、Godotの互換性レンダラーは、名前の通りに高い互換性を持つ反面、Unityほどの表現力を持ちません。そこで、古の技を使うことにします。

ここでは、グラデーションを施したテクスチャーでいい具合にBlenderで貼り付ける手順を備忘録がてら、まとめます。

# モデルの作成

モデルもBlenderで作れますが、今回はMagica Voxelで作成したモデルを、objでエクスポートしたものを使います。

草のオブジェクト

Magica Voxelで、色もざっくりと塗っています。この色を、グラデーションに割り当てていきます。
