
# 頂点カラーにAOを焼き付ける

こちらの記事で、シェーダーで調整したAOを、テクスチャーにベイクする方法を紹介しました。しかし、この方法で綺麗な影を作るには、512x512ピクセル程度のテクスチャーが必要だとわかりました。そこで、デクスチャーを使わなくても済むように、Blenderで頂点カラーにベイクする方法を調べました。

# 手順概要

頂点カラーは、メッシュのColor Attributeに保存できます。シェーダーで焼き付けたい影の設定をしたら、メッシュにベイク先になるColor Attributeを追加して、描画先に指定したら、ベイクします。
